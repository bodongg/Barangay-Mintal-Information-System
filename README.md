# Barangay-Mintal-Information-System
The Barangay Information System is a web-based platform designed to improve the management of barangay records, services, and community information through a centralized digital system. The system provides tools that allow barangay officials to manage resident records, process certificate requests, record blotter incidents, publish announcements, and monitor barangay activities efficiently. Residents can also access the system through an online portal where they can request certificates, view announcements, and obtain important community information without the need to visit the barangay hall.
